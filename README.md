# Dennis Johansson Lloyd - Modern Portfolio

A minimal, high-performance portfolio website built with Next.js 14 and Tailwind CSS.
Designed with a focus on typography, negative space, and subtle interactions.

## Features

- **Minimalist Aesthetic**: Dark mode (Zinc-950), clean lines, and sans-serif typography.
- **Micro-interactions**: Subtle hover states and fade-in animations using Framer Motion.
- **Responsive**: Fully optimized for mobile and desktop.
- **Tech Stack**: Next.js (App Router), React, Tailwind CSS, TypeScript.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Docker

Build and run with Docker:

```bash
docker build -t dennis-cv-modern .
docker run -p 3000:3000 dennis-cv-modern
```
